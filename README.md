# Dynamic Product Card Component

A premium, industry-grade product card component built with **React.js** and **CSS**. This project demonstrates a responsive, interactive, and visually polished UI suitable for modern e-commerce platforms.

## Features

*   **Responsive Design**: Seamlessly adapts from a fluid mobile layout to a fixed grid on desktop.
*   **Interactive Elements**:
    *   "Add to Cart" button with temporary success state and animation.
    *   Heart icon allowing users to toggle their "Favorite" status.
    *   Smart discount badges that calculate percentage off automatically.
*   **Premium UI Details**:
    *   Glassmorphism effects on floating action buttons.
    *   Modern **Inter** typography.
    *   Realistic shadow handling and hover lift effects.
*   **Localization**: Prices formatted in **Indian Rupees (‚Çπ)**.

## Getting Started

Follow these instructions to get the project up and running on your local machine.

### Prerequisites

Make sure you have the following installed on your laptop:
*   **Node.js** (v14 or higher) - [Download here](https://nodejs.org/)
*   **Git** - [Download here](https://git-scm.com/)

1.  **Clone the Repository**
    Open your terminal or command prompt and run:
    ```bash
    git clone https://github.com/YOUR_USERNAME/dynamic-product-card.git
    ```

2.  **Navigate to the Project Directory**
    ```bash
    cd dynamic-product-card
    ```

3.  **Install Dependencies**
    This installs all the necessary libraries (React, Vite, etc.):
    ```bash
    npm install
    ```

4.  **Start the Development Server**
    ```bash
    npm run dev
    ```

5.  **View the App**
    Look for the "Local" URL in your terminal (usually `http://localhost:5173/`) and open it in your web browser.
